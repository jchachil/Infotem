TUIWebApp

  • Pour lancer la solution il faut choisir les projets TUIFrontEnd.Web TUIBackEnd.Web comme projet de démarrage.
  • Le projet TUIFrontEnd.Web est en MVC , il appelle les services de la WEBAPI.
  • Le projet TUIBackEnd.Web est en WebAPI, il appel les méthodes de la couche TUIBackEnd.BLL en utilisant l'injection de dépendence.
  • L'injection de dépendence a été réalisé par le framework Unity
  • La couche TUIBackEnd.DAL effectue l'accès au donnes en utilisant Entity Framework Database First
  • Le projet TUIBackEnd.Test contient les tests unitaires
